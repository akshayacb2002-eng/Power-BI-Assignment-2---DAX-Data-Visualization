# Power-BI-Assignment-2---DAX-Data-Visualization
Power BI project showcasing data transformation, data modeling, DAX calculations, and interactive sales analysis dashboard.

## Project Overview
This Power BI project focuses on data transformation, data modeling, DAX calculations, and dashboard creation to analyze e-commerce sales performance.

## Tools Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

## Key Features
- Data Cleaning and Transformation
- Data Modeling and Relationships
- KPI Cards for Sales, Target, Orders, and Profit
- Interactive Dashboard Visualizations

## Dashboard Visuals
- Year-to-Date Sales Trend
- Monthly Sales Trend
- Sales Target Achievement by Category
- Profit vs Quantity by Sub-Category
- Max Profit Margin by Sub-Category
- Sales Distribution by Sub-Category (Treemap)
- Geographic Sales Analysis
- Order Count Analysis by State
- Minimum Target by Category
- Sales Performance Matrix

## Key Insights
- Total Sales: ₹431.5K
- Sales Target: ₹435.9K
- Total Orders: 500
- Average Profit: 15.97
- Highest sales were generated by Printers and Bookcases.
- Sales peaked during April.
